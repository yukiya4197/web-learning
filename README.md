# Web Learning Guide

このリポジトリは、Webページの作り方を初心者向けに紹介する簡単な教材です。

主に以下の内容を扱っています。

- Webサイトを作る方法（GUIツール / HTML・CSS）
- HTMLの基本タグ
- CSSの基本
- 開発ツール（VS Codeなど）
- GitHub Pagesでの公開
- 生成AIを活用した学習方法

プログラミング経験がほとんどない人でも、Web制作の基本を理解できることを目的としています。
